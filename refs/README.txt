reference signal file naming convention is as follows:

(freq band)_(amp index)_(waveform index).dat

where;
freq band       = L or X
amp index       = 0 - 1
waveform index  = 1 - 14

available amp index:
0 = medium power amp
1 = high power amp

available waveform index:
LFM              NLFM
0.5us   = 1      0.5us   = 8
1.0us   = 2      1.0us   = 9
3.0us   = 3      3.0us   = 10
5.0us   = 4      5.0us   = 11
10.0us  = 5      10.0us  = 12
15.0us  = 6      15.0us  = 13
20.0us  = 7      20.0us  = 14
